All of them are phases
* compile
* test
* verify

Compile -> Test -> Verify

To compile a project
mvn compile

To execute the test in console line with maven
mvn test

To use tags and execute just one specific tag in maven (the -D is to pass parameters)
mvn test -D cucumber.filter.tags="@AddPlace"

Is executed after all the tests ran
mvn test verify

Cucumber Reporting

To open a java file/.jar
java -jar

To start Jenkins from command line using .war file
java -jar jenkins.war -httpPort=9090
java -jar jenkins.war --enable-future-java --httpPort=9090
